# Modelo de Conhecimento

O SDM organiza o conhecimento do planejamento de mídia em quatro grandes domínios.

---

# 1. Campanha

Representa o problema de negócio.

É composta por:

- anunciante;
- objetivo;
- orçamento;
- período;
- praça;
- público.

---

# 2. Público

Representa quem será impactado.

É composto por:

- universos;
- segmentos;
- interesses;
- jornadas.

O modelo deve ser totalmente configurável.

---

# 3. Inventário

Representa onde a campanha pode ser executada.

É composto por:

- canal;
- ambiente;
- plataforma;
- formato;
- tecnologia;
- modalidade;
- unidade;
- indicadores.

Toda caracterização do inventário deve ser parametrizável.

---

# 4. Conhecimento

Representa as relações entre campanha e inventário.

Inclui:

- pesos;
- aderência;
- afinidade;
- cobertura;
- frequência;
- restrições.

Essas relações alimentam os motores de decisão.
