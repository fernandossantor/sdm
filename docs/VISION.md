# Visão do Produto

## Propósito

Construir um sistema especialista capaz de apoiar profissionais de mídia durante todas as etapas do planejamento estratégico.

O SDM deve combinar conhecimento estruturado, regras de decisão, simulações e inteligência artificial para produzir recomendações transparentes e auditáveis.

---

## Visão de longo prazo

O SDM deverá evoluir para uma plataforma SaaS destinada a:

- agências;
- anunciantes;
- instituições públicas;
- universidades.

---

## Princípios

- Transparência das decisões.
- IA explicável.
- Separação entre interface e motores.
- Base de conhecimento configurável.
- Independência tecnológica da interface.
